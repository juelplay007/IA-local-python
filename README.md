# 🐍 PyTutor AI - IA Local para Ensino de Python

Uma assistente de programação baseada em IA que roda **completamente offline** no seu computador, projetada para ensinar Python de forma interativa, tirar dúvidas, corrigir exercícios e sugerir exemplos personalizados.

> ✅ Sem necessidade de internet  
> ✅ Privacidade total (nenhum dado sai da sua máquina)  
> ✅ Gratuito e open-source

---

## ✨ Funcionalidades

- 🤖 **Chat interativo** – Tire dúvidas sobre sintaxe, lógica, bibliotecas, etc.
- 📝 **Correção de código** – Envie um exercício resolvido e receba feedback.
- 🧠 **Geração de exercícios** – Peça desafios personalizados por nível (iniciante, intermediário, avançado).
- 📚 **Explicações passo a passo** – A IA descreve o funcionamento de qualquer código Python.
- 🔍 **Debug assistido** – Cole um código com erro e a IA explica a causa e a solução.
- 💾 **Histórico local** – Salva automaticamente suas conversas (opcional).

---

## 🖥️ Requisitos

- Python 3.10 ou superior
- 8 GB de RAM (recomendado 16 GB para modelos maiores)
- 5 GB de espaço livre em disco
- (Opcional) GPU com suporte a CUDA para execução mais rápida
